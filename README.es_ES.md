**x-panel-ce** — Un panel de control web avanzado de código abierto, diseñado para gestionar el servidor Xray-core. Proporciona una interfaz fácil de usar para configurar y monitorizar diversos protocolos VPN y proxy.

> [!IMPORTANT]
> Este proyecto es solo para uso personal y comunicación. Por favor, no lo uses con fines ilegales ni en entornos de producción.

## Inicio rápido

```sh
bash <(curl -Ls https://raw.githubusercontent.com/hehelove/x-panel-ce/master/install.sh)
```

Para la documentación completa (instalación, configuración, SSL, bot de Telegram, API, Docker, etc.), consulta [README.md](./README.md) o [docs/ROADMAP.md](./docs/ROADMAP.md).
